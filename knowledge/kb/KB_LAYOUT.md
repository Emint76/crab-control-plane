# KB_LAYOUT

## Layout

- `sources/`
- `knowledge/`
- `collections/`

## Discipline

KB stores sanctioned assets only.

### Source-bearing asset minimum
- canonical pointer
- retrieval status
- retrieval timestamp
- content type
- stable representation
- human identifier

### Knowledge asset minimum
- clear knowledge role
- intelligible content
- source linkage where relevant
- review outcome suitable for placement
