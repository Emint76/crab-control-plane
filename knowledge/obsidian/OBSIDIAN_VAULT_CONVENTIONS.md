# OBSIDIAN_VAULT_CONVENTIONS

Obsidian is the semantic plane.

## Minimum frontmatter

```yaml
---
id: ""
title: ""
note_type: ""
status: ""
created: ""
updated: ""
tags: []
source_links: []
conceptual_links: []
---
```

## Allowed note types

- source-note
- permanent-note
- concept-note
- comparison-note

## Rule

Do not use Obsidian as a task queue or workflow state machine.
