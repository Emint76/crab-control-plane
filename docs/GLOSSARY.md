# GLOSSARY

## Control plane
Versioned layer of policies, contracts, templates, and architecture docs.

## Operational plane
Workflow state: intake, queues, statuses, handoffs.

## Semantic plane
Durable human-readable notes, concepts, and semantic links.

## Sanctioned asset
Artifact approved for KB placement.

## Task packet
Structured request for subtask execution.

## Result packet
Structured response from subtask execution.

## Placement
Decision about where an artifact belongs.

## Provenance
Minimum origin and capture information needed for trustworthy reuse.
